About

Text analysis web app. Provides the user with an estimated grade level for a piece of text. English only. Currently ignores words not found in the dictionary ('cmudict' from the nltk corpus).

Planned additions: unknown word handling, additional formulae (Gunning fog, SMOG, Fry readability, Coleman-Liau).

If you have any questions, please email me.

This web app utilizes the following:
-Django
-PostgreSQL
-NLTK (Natural Language Toolkit)
-gunicorn
-the Textrazor API

Automated Readability Index

Quoth Wikipedia:

"The automated readability index (ARI) is a readability test for English texts, designed to gauge the understandability of a text. Like the Flesch–Kincaid grade level, Gunning fog index, SMOG index, Fry readability formula, and Coleman–Liau index, it produces an approximate representation of the US grade level needed to comprehend the text."
